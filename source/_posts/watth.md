# 0.

也许我们的思考总是在经历一些事后被我们察觉，于是我们想当然地认为这些事是思考的缘由，换句话说，我们自以为思考是被启发出来的。但是，如果转换一下角度，从偶然性抽离出来再思考这个问题——我们的思考也许是个必然？

写这篇文章是因为读到了不感兴趣的文章，虽然这篇的作者我有些好感，起码这个作者在这之前一直占据着“思考的缘由”的位置上，但是题材我真的极其不感兴趣。于是有了这些问题。

# 1.

> 哲学的意义，究竟在于回答，还是在于思考的过程。

当我们把意义放置在答案上——哲学一下子被固定住了。哲学家们千百年来用思考为其赋予的生命变成了一摊死水。

哲学本身就很难有一个明确的答案，历代哲学家用不同的角度和方法论证证这个世界。
谁也没有成功，从苏格拉底到齐泽克，他们各有各的局限性。

但是，哲学毫无意义吗？

在哲学家不断尝试阐述世界时，他们带来的尝试的方法论，仍旧闪耀在其他学科领域。

--逻辑学，数学，物理学乃至玄学物理学。

那么，重要的，还只是答案吗？

# 2.

再将思考的过程放在哲学的意义上。那么哲学家们所研究的就不应是世界如何，而是*我们如何*。

很可惜的是，有相当一部分哲学家并不会会去思考关于思考本身的问题。

**前苏格拉底哲学家**（公元前7-5世纪）基本**未涉及**对思考的思考，以至于他们缺乏对认知个体的系统性认知。
就算是对于近现代发展出来的分析哲学来说，“我”这一字，本身就是非法的。

可见，哲学发展的主线，从来不在于思考本身。

# 3.

从头来过。

哲学，希腊语中又有**爱智慧**的意思，它诞生于人类的好奇心。人类迫切地将世界纳入自己构建的体系当中，于是他们用自己有限的知识推断完整的宇宙。

哲学最开始就是为了将世界正确地放在自己的知识当中。

此事在精神分析中的**镜子阶段**亦有记载。

# 4.

镜子阶段的理论来源于一次动物学实验，结论是

> 半岁大的儿童与同龄的黑猩猩的不同，即在于前者会变得**痴迷于其镜中的映像**，并且欢欣雀跃地把它**接纳为其自身的形象**，而黑猩猩则会迅速认识到那一形象是虚幻的，并且丧失对它的兴趣。

镜子阶段阐明了人类之所以为人的一个标志，将外界形象认同为完整的自我。在此阶段，我们将自我的形象误认为真实的自我。

哲学的一个重要命题就是如何*认识世界*。正是镜子阶段的存在，我们将外界形象认同为自我，这种全能感让我们认为我有权力理解世界的运行规则。所以，我们开始好奇这世界万物，并尝试给予他合理的说法。

# 5.

关于“我”的讨论中，从尼采对形而上学语法的批判开始，在**结构主义**运动中福柯《词与物》的发表以后，主体不再被认为是进行思考的主体，仅仅是思考发生的场所。换句话说，主体只是思考需要的条件。

~~（当然这个依旧有争论，比如分析哲学仍然认为主体是实体，不过本人更倾向于结构主义和现象学，也就是本文所述的观点）~~

思考既必然，也不属于我。

因此，哲学的意义随世界而变，它的形象是最终真理，即便他本身只能无限逼近，永无到达那日。我们始终不是镜子那一头的形象。

# 6.

在现象学看来，心灵或意识指向、呈现或关于某物的能力，是心理现象区别于物理现象的核心特征，这种能力被称为意向性。思考某物就是心灵意向性的一种体现。只有当被呈现的那物可以被感知到时，我们才能说“我正在思考”。

# 7.

可是，在我们解构问题以后，我们仍旧未找出答案。但我们现在可以确定思想有必然与客观存在性，但是其客观性是相对的，哲学是爱真理的学科，客观并不能代表他的正确性。一个哲学命题，如何被思考与被讨论，必须需要一个媒介人为地将其从脑海中解放出来。只有当这种思想赤裸地展现在世界的真实当中，人们达到了思想中所预想的结果时，人们的认识才被证实了。

而哲学，正是这一旅程的指南书；他不为后人提供结果，只是证明曾经有个人正在思考。
