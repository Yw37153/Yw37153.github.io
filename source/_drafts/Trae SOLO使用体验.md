大家好我是从来不带节奏的~~血狼破军~~，今天为大家带来一款 **AI 原生工作台**，提供网页版、桌面版和移动版，并设有为不同用户群体设计的 Work 与 Code 双模式。

先说结论，泛用性强，高难弱，无法在工作台里人为更改文件（也许是因为把这个活和IDE做了切分)，适用于需要一些【简单】修改文件，【重复】工作量较大的人群，需求【多端】同步，但文字生成能力还可以，总体归为中杯上。

这款工作台非常之适合一些轻度使用AI的用户，功能不算强大但足够使用，同时图形化页面给一些配置和交互工作减轻了不少压力。下面我将介绍一些常规用法和使用中遇到的问题。

# 安装
安装步骤非常简单，和其他PC软件相同。只需要进入官网，点击下载中心，点击下载TRAE SOLO CN后运行`.exe`即可。
