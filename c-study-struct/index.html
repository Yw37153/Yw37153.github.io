Struct

概念

C 数组允许定义可存储相同类型数据项的变量，结构是 C 编程中另一种用户自定义的可用的数据类型，它允许您存储不同类型的数据项。

定义结构体

struct tag: 新结构体名称（类似int float）
member-list:结构体成员，可以理解为属性。
viariable-list:结构变量。相当于先写了一个新的结构体tag，然后说明variable-list是tag类型。

  • 关键字 typedef

此时tag和variable-list为新的新结构体名称，不过需要注意的是，typedef必须要有末尾类型名，而前面的tag类型名则可以省略。
