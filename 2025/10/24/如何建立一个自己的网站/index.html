1.准备工作 别扔盘的根目录底下：）

1.3 为你的博客管理文件夹安装hexo框架
经过多次实验，，别扔根目录（要扔的话要给权限好麻烦（）

  • 右键心仪的文件夹，用git bash打开，输入代码，全局安装hexo
  • 检查是否成功
  • 初始化hexo（这步要确认文件夹为空,同时保正可以与github正常连接）
  • 本地部署

完成这步就可以用ctrl+左键打开一个有4000的链接，这样已经可以看到网站雏形了。
接下来我们要将本地上传到github，这样才能让大家都看到你。

2.部署到github

  • 创建仓库
    Github中new一个repository
    Repository name [用户名.github.io]

  • 本地配置SSH Key
    在git bash页面分别输入以下命令

    完成后右键打开<id_rsa.pub>文件并复制里面的全部内容

  • 添加SSH Key到Github
    https://github.com/settings/ssh/new
    New即可

  • 同步SSH Key到本地git

  • 修改Hexo配置文件
    在_config.yml中#Deployment下替换：

#Deployment
##Docs: https://hexo.io/docs/deployment.htm
deploy:
type: git
repo: git@github.com:XXXXX/XXXXX.github.io.git
branch: main
