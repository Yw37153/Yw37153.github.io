## 如果幸运的话，你会看到我。

> 少年不识愁，惟愿乐相随。

喜欢写点日记，写点不知所云的文字，写点我的观点和看法。我想让自己更加**赤诚**地活在这个世界上。

想让更多人看见我，看见我的人生多么精彩，看见我先进的观念并认同它们。

更希望大家可以陪伴在我的身边，无论是爱情、友情。

由于你正在使用 **Hexo + Butterfly**，这个主题的自由度极高，但也容易让人在细节里“迷失”。为了避免漫无目的地修改，我为你整理了一份从**视觉基础**到**交互体验**的 TODO 计划表。

这份计划表按照先整体、后局部的逻辑排列，建议你按顺序推进：

---

## 🎨 第一阶段：确定视觉基调 (The Foundation)
在动代码之前，先确定“脸面”：
- [x] **确定配色方案**：选定 1 个主色调（Theme Color）和 1 个辅助色。
    * *提示：在 `_config.butterfly.yml` 的 `theme_color` 中修改。*
- [x] **字体风格统一**：
    * [x] 选定全局字体（推荐 `Noto Sans SC` 或 `PingFang SC`）。
    * [x] 选定代码字体（推荐 `Fira Code` 或 `JetBrains Mono`）。
- [/] **深色模式校准**：检查在 Dark Mode 下，你的 Logo 和某些自定义图片是否清晰。

---

## 🖼️ 第二阶段：多媒体与品牌感 (Branding)
利用你已有的 S3 图床，把核心视觉元素固化下来：
- [x] **精选 Banner 图**：
    * [x] 准备一张高清的首页大图（`default_top_img`）。
    * [/] 为不同分类/标签准备不同的 `top_img`（可选，增加差异化）。
- [x] **完善 Favicon & Logo**：
    * [x] 确保浏览器标签页的小图标（Favicon）在各种尺寸下都清晰。
- [x] **头像与社交卡片**：
    * [x] 检查侧边栏 Avatar 显示（确保你已经关掉了那个“旋转”效果）。
    * [x] 配置底部的社交图标链接。

---

## 🛠️ 第三阶段：布局与功能瘦身 (Layout & Functions)
Butterfly 默认功能很多，建议做减法：
- [x] **导航栏 (Nav) 整理**：
    * [x] 检查分类 (Categories)、标签 (Tags)、关于 (About) 页面是否已创建。
    * [x] 给菜单项配上合适的 Font Awesome 图标。
- [x] **侧边栏 (Aside) 优化**：
    * [x] 隐藏掉不常用的组件（如：公告栏、最近评论、标签云，按需保留）。
    * [x] 调整组件的显示顺序（推荐：个人信息 > 公告 > 最新文章）。
- [x] **页脚 (Footer) 定制**：
    * [x] 修改版权信息。
    * [x] 添加备案号（如果需要）或运行时长统计。

---

## ⌨️ 第四阶段：代码与内容呈现 (Content UI)
提升读者的阅读体验：
- [x] **代码块美化**：
    * [x] 在配置中选择喜欢的主题（`highlight_theme`）。
    * [x] 开启代码行号、复制按钮。
- [x] **文章封面 (Cover) 策略**：
    * [x] 统一文章封面的长宽比。
    * [x] 决定是使用“左图右文”还是“交替排列”模式。
- [x] **排版细节**：
    * [x] 开启文章末尾的版权协议声明（`post_copyright`）。
    * [x] 开启打赏功能或打上特定的 Tag。

---

## 🚀 第五阶段：性能与收尾 (Performance & Polish)
设计不仅要好看，还要快：
- [x] **引入搜索功能**：配置本地搜索（Local Search），让内容可查。
- [ ] **图片懒加载**：在 `lazyload` 配置中开启，提升首屏速度。
- [ ] **最后检查**：
    * [ ] 执行 `hexo clean`。
    * [ ] 手机端打开浏览器，检查移动端适配是否错位。
