第二十章 曲线积分

一、第一型曲线积分（对弧长的曲线积分）

1. 参数化计算公式

设曲线 LLL 的参数方程为：

L:{x=x(t)y=y(t)t∈[α,β]L: \begin{cases} x = x(t) \\ y = y(t) \end{cases} \quad t \in [\alpha, \beta]L:{x=x(t)y=y(t)​t∈[α,β]

则第一型曲线积分计算公式为：

∫Lf(x,y)ds=∫αβf(x(t),y(t))x′2(t)+y′2(t)dt\int_L f(x, y) ds = \int_{\alpha}^{\beta} f(x(t), y(t)) \sqrt{x'^2(t) + y'^2(t)} dt∫L​f(x,y)ds=∫αβ​f(x(t),y(t))x′2(t)+y′2(t)​dt

2. 几何意义

以 LLL 为准线，母线平行于 zzz 轴，截取 0≤z≤f(x,y)0 \le z \le f(x, y)0≤z≤f(x,y) 的柱面部分的面积。

3. 示例与练习

  • 半圆周示例：

    L:{x=acos⁡ty=asin⁡t0≤t≤πL: \begin{cases} x = a\cos t \\ y = a\sin t \end{cases} \quad 0 \le t \le \piL:{x=acosty=asint​0≤t≤π

    计算 ∫L(x2+y2)ds\int_L (x^2 + y^2) ds∫L​(x2+y2)ds：
    ∫L(x2+y2)ds=∫0πa2⋅adt=a3π\int_L (x^2 + y^2) ds = \int_0^{\pi} a^2 \cdot a dt = a^3 \pi∫L​(x2+y2)ds=∫0π​a2⋅adt=a3π

  • 交线圆示例：已知曲线为球面与平面的交线圆：

    {x2+y2+z2=a2x+y+z=0\begin{cases} x^2 + y^2 + z^2 = a^2 \\ x + y + z = 0 \end{cases}{x2+y2+z2=a2x+y+z=0​

    利用对称性计算：
    ∫Lx2ds=13∫L(x2+y2+z2)ds=13∫La2ds=13a2⋅2πa=23πa3\int_L x^2 ds = \frac{1}{3} \int_L (x^2 + y^2 + z^2) ds = \frac{1}{3} \int_L a^2 ds = \frac{1}{3} a^2 \cdot 2\pi a = \frac{2}{3}\pi a^3∫L​x2ds=31​∫L​(x2+y2+z2)ds=31​∫L​a2ds=31​a2⋅2πa=32​πa3

二、第二型曲线积分（对坐标的曲线积分）

1. 定义与记号

力和变力做功的积分：

∫LP(x,y)dx+Q(x,y)dy\int_L P(x, y)dx + Q(x, y)dy∫L​P(x,y)dx+Q(x,y)dy

其中 PPP 和 QQQ 是关于正方向的速率分量。若 LLL 封闭，则记作：
∮LPdx+Qdy\oint_L Pdx + Qdy∮L​Pdx+Qdy

2. 方向性（与方向有关）

∫ABPdx+Qdy=−∫BAPdx+Qdy\int_{AB} Pdx + Qdy = -\int_{BA} Pdx + Qdy∫AB​Pdx+Qdy=−∫BA​Pdx+Qdy

3. 参数化计算公式

设 LLL 的起点到终点参数方程为：

L:{x=φ(t)y=ψ(t)t:α→βL: \begin{cases} x = \varphi(t) \\ y = \psi(t) \end{cases} \quad t: \alpha \to \betaL:{x=φ(t)y=ψ(t)​t:α→β

对应的起点和终点坐标为 A(φ(α),ψ(α))A(\varphi(\alpha), \psi(\alpha))A(φ(α),ψ(α)) 和 B(φ(β),ψ(β))B(\varphi(\beta), \psi(\beta))B(φ(β),ψ(β))。
∫LP(x,y)dx+Q(x,y)dy=∫αβ[P(φ(t),ψ(t))φ′(t)+Q(φ(t),ψ(t))ψ′(t)]dt\int_L P(x, y)dx + Q(x, y)dy = \int_{\alpha}^{\beta} \left[ P(\varphi(t), \psi(t))\varphi'(t) + Q(\varphi(t), \psi(t))\psi'(t) \right] dt∫L​P(x,y)dx+Q(x,y)dy=∫αβ​[P(φ(t),ψ(t))φ′(t)+Q(φ(t),ψ(t))ψ′(t)]dt

  • 直角坐标直接代入法：若 y=f(x)y = f(x)y=f(x)，自变量从 x1→x2x_1 \to x_2x1​→x2​：则将 y=f(x)y = f(x)y=f(x) 和 dy=f′(x)dxdy = f'(x)dxdy=f′(x)dx 直接代入积分式中，转化为对 xxx 的定积分。例如 y=2x−1y = 2x - 1y=2x−1，则 y′=2y' = 2y′=2，dy=2dxdy = 2dxdy=2dx。

三、格林公式 (Green’s Theorem)

∮LPdx+Qdy=∬D(∂Q∂x−∂P∂y)dxdy(逆时针方向)\oint_L Pdx + Qdy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy \quad (\text{逆时针方向})∮L​Pdx+Qdy=∬D​(∂x∂Q​−∂y∂P​)dxdy(逆时针方向)

四、验证积分与路径无关并求值

1. 路径无关的条件

验证区域内是否满足偏导数相等：

∂P∂y=∂Q∂x\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}∂y∂P​=∂x∂Q​

2. 求值方法（分段积分法）

通常将积分路径拆分为水平 + 竖直两段：

  • 水平段：y=Cy = Cy=C（常数），⇒dy=0\Rightarrow dy = 0⇒dy=0

  • 竖直段：x=Cx = Cx=C（常数），⇒dx=0\Rightarrow dx = 0⇒dx=0

五、利用格林公式计算（开口曲线的补全法）

  1. 补全图形：将开口曲线 →\to→ 封闭曲线（添加辅助线段）。

  2. 应用格林公式：

    • 判断边界方向：顺时针还是逆时针（通常逆时针为正向）。
    • 计算偏导数 ∂Q∂x\frac{\partial Q}{\partial x}∂x∂Q​ 和 ∂P∂y\frac{\partial P}{\partial y}∂y∂P​ 并代入公式。
    • 提常数。
    • 计算二重积分 ∬DdA\iint_D dA∬D​dA（若 DDD 是圆，积分结果即为圆的面积）。

  3. 减去辅助线：把补全时多出来的边界积分部分减掉。整理
